Building Blocks

HTML

The Hypertext Markup Language was and still is a fundamental component to Websites. HTML is comprised of many elements that can be used to render unique content on a browser. Each Element ascribes to the "Box-model concept" which simply means that each element is rendered as a rectangle on the web page. The rectangle consists of four layers; margin, border, padding and the actual content layer which is what is visible to the viewer. The other three layers are abstract however they play a key role in positioning and styling the rendered content.

CSS

Cascading Style Sheet is developed using HTML and is solely responsible for the presentation of almost all websites. An example of a presentation aspect of a web page would be background-color. A selector is used to reference the HTML element in question then a property of either one of the four layers we have mentioned above is invoked to which a value is assigned. As its name suggests this technology cascades the styling effect to all children elements.
